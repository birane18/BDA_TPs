Exercice 4 : Algorithmes de Normalisation (Python)

Ce sous-dossier contient l'implémentation en Python des algorithmes d'analyse et de normalisation de schémas relationnels vus en cours de Bases de Données Avancées (BDA).

Structure des fichiers
- algorithmes.py: Contient l'implémentation pure des 14 fonctions demandées (calcul de fermeture, détection de clés candidates, vérification BCNF, algorithme de décomposition).
-main.py : Script principal instanciant les données d'exemple du TP (relations et dépendances fonctionnelles) et exécutant les algorithmes pour démonstration.

Comment exécuter le code ?
Assurez-vous d'avoir Python 3 installé. Le code n'utilise que la bibliothèque standard (itertools).

Placez-vous dans ce dossier depuis votre terminal et lancez :
```bash
python3 main.py
